# 佟筱钰 — 个人简历

一页式在线简历，使用纯 HTML/CSS 构建，托管于 GitHub Pages。

## 访问

🔗 部署后可通过 `https://<your-username>.github.io/<repo-name>` 访问

## 文件结构

```
index.html   ← 完整单页简历
Profile.png  ← 头像照片
README.md    ← 本文件
```

## 部署步骤

1. 创建 GitHub 仓库（建议命名：`resume` 或 `cv`）
2. `git remote add origin https://github.com/<用户名>/<仓库名>.git`
3. `git push -u origin main`
4. 仓库 Settings → Pages → Branch: main → / (root) → Save
